# Description
___
Birzha is a crypto market that allows its users to generate nft collection using AI and trade
directrly on the market via instant sell/buy and auctions.

# Setup
___
Clone the repository, run `python envgen.py` to set up neccessary enviromnents.

To build the docker-compose project run `make dev` for developer build, `make prod` for production build.

You can configure `.env` file to set all variables that you need.
